# Project Memory — ADN Ninja Warrior

## Core
Mobile-first app for kids at obstacle course gym. Dark theme #121212 + neon green + electric blue accents. All UI copy in castellano (es-AR).
Two-tab bottom nav: "Mi Camino Ninja" (/) and "Reporte Mensual" (/report).
No competition/rankings — gamification based purely on attendance & discipline.
Obstacle illustrations: kid-friendly hand-drawn marker/crayon style on flat #121212 background, with neon green + electric blue accents. Keep style consistent across all obstacles.
